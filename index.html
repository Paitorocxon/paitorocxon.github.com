Who am i?


About me

I’m Paitorocxon, a software developer and security enthusiast who enjoys experimenting with unusual ideas and technologies.

Around 2013 I started working on C.A.R.A (i), a relatively simple AI at first. I began with speech recognition, then added a dynamically learning language library, and over time it became possible to have fairly natural conversations with it. C.A.R.A no longer exists in its original form, but the project strongly influenced how I think about language, interaction and tools today.

Selected Projects

C.A.R.A (i)

An early AI assistant experiment started around 2013, combining speech recognition with a dynamically learning language library. The goal was to make it possible to talk to the system in a more open, conversational way instead of using fixed commands.

NeuralNetwork

A small neural network implementation written in C#. It focuses on being lightweight and easy to integrate into .NET projects, making it suitable for experiments with simple machine learning tasks without relying on large external frameworks.

Plain-GDI-3D

A prototype 3D engine built on top of classic GDI+ drawing. It renders basic 3D scenes using only 2D drawing primitives, mainly as a playground to explore engine concepts and low-level rendering without a dedicated 3D framework.
